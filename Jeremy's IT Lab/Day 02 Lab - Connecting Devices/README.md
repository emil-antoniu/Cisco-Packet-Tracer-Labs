# Day 2 Lab 

## Overview
This lab walks through connecting basic network devices using Cisco Packet Tracer.

## Key Activities 
- Select appropriate cables based on the device types and ports (e.g., straight-through for same devices types, crossover for different device types).
- Learn about Auto MDI-X - the succesor to MDI and MDI-X. It is capable of automatically detecting the receive and transmit cable pairs and swapping them internally as needed. This lab highlights the old-school way of connecting devices.
- Learn about a multimode vs. singlemode fiber use case - multimode is better suited for short-distance transmission (<400m), while singlemode is better suited for long-distance transmission.
